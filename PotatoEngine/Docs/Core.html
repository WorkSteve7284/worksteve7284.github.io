PotatoEngine Core Libraries

The PotatoEngine Core Libraries are the five C++ libraries designed to give you the ability to—with relative ease—create a functioning game. Some of the more advanced functionality you may be familiar with is included in other libraries.


Requirements for PotatoEngine Core Libraries

The Core Libraries are designed to be somewhat inter-dependant. The PotatoEngine library is completely self-sufficient, and doesn't need any other libraries except the C++ standard libraries. Physics relies on PotatoEngine and Input, Rendering, and UI rely on Physics, PotatoEngine, and GTK (an external library)
