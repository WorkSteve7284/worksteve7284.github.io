PotatoEngine Library

The PotatoEngine library contains everything you need to have a very basic game. If you don't use this library, you basically aren't using PotatoEngine.


The PotatoEngine library contains five main classes, Master_Scene, Scene, Object, Component, and Event.

There is only one Master_Scene, which handles all of the other Scenes. Scenes contain Objects, which contain Components.

Components are where your code goes.

Events are broadcast over the event system to whichever Components are listening to them.
