Component

Components are the most basic component of a project in PotatoEngine. a Component contains your code, so nearly all of your code will be within them.


Object *object

object is a pointer to the Object which owns this Component. object is set during the creation of a Component.


void Start()

Start is called when a Scene is first activated or a Component or Object is created during runtime.


void Update(double delta_time)

Update is called as often as the engine can, as long as the time between calls is greater than 1ms. delta_time is equal to the time since the last call of Update.


void Fixed_Update(double delta_time)

Fixed_Update is called as close to a fixed time step as possible. By default, this value is every 1/60th of a second. delta_time is the time since the last call of Fixed_Update


Usage

This script is an example of how you might write a Component. It prints the number of Updates and the run time to console.


            // example.hpp

            // Includes
            #include <iostream>
            #include <string>
            #include "PotatoEngine.h"

            class Example_Component : public Component {            
                int updates = 0;
                double fixed_time = 0;

                void Start() {
                    object->name = "Example Object";
                }

                void Update() {
                    updates++;
                    std::cout << updates << std::endl;
                }
