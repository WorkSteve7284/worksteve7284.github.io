PotatoEngine.h

The PotatoEngine.h header contains the three basic classes required for a project in PotatoEngine: Component, GameObject, and Scene. Scenes contain GameObjects, GameObjects contain Components, and Components contain your code.


Component

A Component is the basic block of a PotatoEngine Project. If you want to create custom code for your project, you create a new class which inherits the Component class.

Component, by default, has just a destructor, Start, Update, Fixed_Update, and owner.


GameObject* owner

owner is a pointer towards the GameObject which owns this Component. It is set when a Component is added to a GameObject (see GameObject)


void Start()

Start is called as soon as a Scene is first loaded. The Scene calls a GameObject's Start, which then calls the Component's Start.


void Start() {
    // Your code here
}

void Update(double delta_time)

Update is run as often as the engine can run it, again through the same Scene→GameObject→Component pipeline. delta_time represents the time since the last execution of Update, and can be useful for realtime applications.


void Update(double delta_time) {
    // Your code here
}

void Fixed_Update(double delta_time)

Fixed_Update is run every n seconds, where n is set in src/main.cpp (default 0.166). Fixed_Update is also run through the Scene→GameObject→Component pipeline. delta_time represents the time since the last execution of Fixed_Update. This may not always be your set time, due to factors such as long computation time.


void Fixed_Update(double delta_time) {
    // Your code here
}

GameObject

A GameObject is the next step up in the structure of PotatoEngine. GameObjects can have any number of Components attached to them.

GameObject has Start, Update, and Fixed_Update functions, which call their respective counterparts in all child Components.


std::vector<std::unique_ptr<Component>> components

components is a list of Components owned by this GameObject.


T* AddComponent(Args&&...args)

AddComponent creates and adds a new component of type T to this GameObject, and returns a pointer to it.

Use this like: gameObjectAddComponent<Transform>( constructors )


template<typename T, typename... Args>
T* AddComponent(Args&&... args) {
    T* comp = new T(std::forward(args)...);
    comp.owner = this;
    components.emplace_back(comp);
    return comp;
}

T* GetComponent()

GetComponent finds the first Component with type T in this GameObject's components vector
